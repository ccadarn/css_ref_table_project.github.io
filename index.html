Common CSS properties

CSS (Cascading Style Sheets) provides a wide range of properties that you can use to style and format HTML elements. Here are some common CSS properties and their descriptions:

Name Description
Color Sets the text color of an element
background-color Sets the background color of an element
font-size Specifies the size of the text
font-family Defines the font used for text
font-weight Sets the boldness of the text
text-align Aligns the text within an element (left, right, center, or justify)
text-decoration Adds decorations to the text (underline, overline, line-through, etc.)
padding Sets the space between the content and the border of an element
margin Defines the space outside an element
border Sets the border properties (width, style, color) of an element
width Specifies the width of an element
height Specifies the height of an element
display Determines how an element is displayed (block, inline, inline-block, etc.)
position Sets the positioning method for an element (static, relative, absolute, fixed)
float Positions an element to the left or right of its container.
clear Specifies whether an element should be moved below floating elements
opacity Sets the transparency of an element
transition Specifies the transition effects for an element (property, duration, timing function)
box-shadow Adds a shadow effect to an element
border-radius Rounds the corners of an element

These are just a few examples of common CSS properties. CSS provides a vast array of properties that allow for precise control over the appearance and layout of web pages. It's important to note that different properties apply to different types of elements, so it's helpful to consult CSS documentation or references for a comprehensive list and usage details of CSS properties.
